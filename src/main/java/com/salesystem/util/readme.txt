В пакете model:
Order - для хранения информации об одном заказе из входного файла.
OrderResult - для хранения итогового результата расчёта стоимости для одной компании.

В пакете parser:
OrderParser - превращает строку из файла в объект Order.

В пакете service:
OrderService - вычисляет цены со скидками.

В пакете util:
FileOrderService - читает/записывает файлы заказов

В пакете manager:
FileOrderManager - координирует работу всех классов

Main - стартовая точка
